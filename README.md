# Gobierno-abierto
Objetivo del proyecto: presentar la relación directa entre la organización, la exposición y el diálogo de la información a nivel municipal en Colombia, así como el aumento de estos indicadores entre 2010 y 2015.
Las tecnologías usadas fueron Tableau Public para crear la visualización interactiva, Stata para obtener los coeficientes de correlación de Pearson y las estadísticas descriptivas, ObservableHQ para documentar los resultados en entorno web, y GitHub para el archivo de las piezas que hacen parte del proyecto.
El enlace del proyecto en Observable es https://observablehq.com/@ojzapata/untitled/2
